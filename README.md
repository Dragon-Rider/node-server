# node-server
A Node.js SFB ( Separation of Front and Back ends ) project for building node server fastly and simply, based on [koa][2].   

## Install
```bash
$ git clone https://github.com/Dragon-Rider/node-server.git   
$ cd  ./node-server    
$ npm install  
$ npm start  
```
## Local Running
```
http://localhost:8080/   
```
